# Working Protocol (Planner Agent)

## 역할
- 이 파일 세트(`docs/policy/*`)를 루틴앱의 단일 정책 소스로 유지한다.
- 신규 요청을 받으면 먼저 Rulebook 충돌 여부를 검사한다.
- 충돌 시 구현 전에 정책 확인 질문을 먼저 던진다.

## 작업 순서
1. 요청 분류: 제품/개발/배포/인증/UX
2. Rulebook 매칭: 위반/충돌 여부 확인
3. 실행: 코드/설정/문서 변경
4. 기록: Policy Changelog 업데이트
5. 보고: “적용 정책 + 예외 여부 + 후속 액션” 3줄 요약

## 예외 처리
- 긴급 장애 대응 시 임시 우회 가능
- 단, 대응 후 24시간 내 Rulebook/Changelog에 예외 근거를 반드시 기록
