# 비용 0원 운영 원칙 (Apple 연회비 제외)

## 고정 원칙
- Apple Developer Program 연회비 외 **추가 결제 금지**
- 유료 플랜 전환/결제 버튼 클릭/카드 등록은 **사전 사용자 승인 필수**
- 과금 가능성이 생기면 작업을 멈추고 먼저 보고

## 서비스별 정책
- Expo/EAS: Free 플랜만 사용 (큐 대기 허용)
- Vercel: Hobby(무료)만 사용
- Supabase: Free 플랜만 사용
- 푸시: 서버푸시 미사용, 로컬 알림 사용

## 경고 조건 (즉시 보고)
- 플랜 업그레이드 권유/강제 메시지 노출
- 무료 한도 임박/초과 경고
- 결제 수단 등록 요청
- 유료 API/서드파티 서비스 도입 필요 상황

## 운영 대응
1. 과금 가능성 감지 즉시 사용자에게 보고
2. 무료 대안 제시
3. 승인 전까지 유료 전환 작업 중단
